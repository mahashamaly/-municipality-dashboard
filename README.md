# تقرير أداء البلدية - Power BI

تحليل شامل لأداء البلدية يغطي جانبين: خدمة الشكاوى والأداء المالي.

## 📊 محتوى التقرير

### صفحة الشكاوى
- إجمالي عدد الشكاوى
- نسبة الشكاوى المغلقة
- متوسط تقييم رضا المواطنين
- متوسط مدة حل الشكوى بالأيام
- توزيع الشكاوى حسب الحي
- توزيع الشكاوى حسب نوع الشكوى

### صفحة الميزانية
- إجمالي الإيرادات
- إجمالي المصروف الفعلي
- إجمالي الميزانية المخصصة
- مقارنة الميزانية المخصصة بالمصروف الفعلي حسب القسم

## 🛠️ الأدوات المستخدمة
- Power BI Desktop
- DAX (لحساب المقاييس مثل نسبة الإغلاق)

## 📁 البيانات
جدولين رئيسيين:
- **الشكاوى**: 1,850 سجل
- **الميزانية_الشهرية**: 240 سجل
